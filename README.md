# Employee-Management-System
The Employee Management System is a Java-based desktop application that manages employee data using MySQL. It allows users to add, update, view, and delete records. Built with Java Swing and JDBC, it provides a simple interface and smooth database interaction for efficient employee management.

🚀 Features

🎬 Splash Screen

Animated startup screen for better user experience

Automatically redirects to login page after a delay
