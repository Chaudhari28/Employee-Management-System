# Employee-Management-System
The Employee Management System is a Java-based desktop application that manages employee data using MySQL. It allows users to add, update, view, and delete records. Built with Java Swing and JDBC, it provides a simple interface and smooth database interaction for efficient employee management.

🚀 Features

🔐 Secure Login System

Validates user credentials using MySQL database

Displays error message for invalid login attempts

🎬 Splash Screen

Animated startup screen for better user experience

Automatically redirects to login page after a delay

🖥️ GUI using Java Swing

Interactive and user-friendly interface

Custom background images and layout

🗄️ Database Integration

Connected with MySQL using JDBC

Stores and verifies user credentials from database

📊 Main Dashboard

Redirects to main screen after successful login
